slipstream-gui

A minimal Windows GUI for running Slipstream and routing system traffic through a local SOCKS5 proxy.

slipstream-gui provides a simple toggle to start/stop the Slipstream client and automatically configure the Windows system proxy.

Features

One-click enable / disable

Runs Slipstream client in the background

Starts a local SOCKS5 proxy on 127.0.0.1:5201

Automatically sets and clears Windows system proxy

Visual connection state (Disabled, Connecting, Enabled)

Detects startup failures and shows an error if the proxy is not reachable
